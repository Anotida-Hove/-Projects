 Gender Inequality & Human Development Analysis
Tools: R · ggplot2 · tidyverse
Overview
An exploratory data analysis investigating the relationship between gender inequality and human development across 191 countries using 2023 UNDP data.
Key Findings

Identified a strong negative correlation between the Gender Inequality Index (GII) and the Human Development Index (HDI) across all development tiers.
Countries with lower gender inequality consistently showed higher human development scores.

What I Did

Cleaned and transformed raw UN datasets using data wrangling techniques (variable renaming, pivot operations, derived variables)
Produced 5 data visualizations including scatterplots, boxplots, and grouped bar charts
Analyzed trends across low, medium, high, and very high human development categories

How to Run

Clone the repo
Open the .Rmd or .R file in RStudio
Install dependencies: tidyverse, ggplot2
Run the script to reproduce all visualizations
