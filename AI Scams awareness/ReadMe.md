Deepfake Aware — AI Ethics Education Website
Tools: HTML · CSS · JavaScript
Overview
A multi-page educational website designed to raise awareness about deepfake scams and AI ethics, built with a focus on accessibility and interactive learning.
Features

Sidebar navigation system across multiple pages
Tabbed content and collapsible discussion prompts
8-question interactive quiz with instant feedback
WCAG 2.1 compliant — keyboard navigation, ARIA roles, screen reader support, and a high-contrast color system

What I Did

Designed and built the full site from scratch using vanilla HTML, CSS, and JavaScript
Used Claude AI to generate and iterate on JavaScript logic, then reviewed, tested, and refined the output to meet accessibility and functionality standards
Verified contrast ratios throughout to ensure readability for all users

How to Run

Clone the repo
Open index.html in your browser — no setup required
